# Summary

Bought a bunch of tiny OLED displays on ebay, none of the cases I could find would fit this model. 
So here is one which snaps together, I included the 123D incase anyone want's to more easily integrate the bezel into another box.

0.96" I2C IIC SPI Serial 128X64 OLED LCD Display SSD1306
They're about 27,3mm x 28mm
